1. System Dynamics Overview

This model represents a coupled socio-ecological system for a pre-industrial rural Bengali village (c. 1000-1800 CE). The core hypothesis is that human cooperation and ecological resource base form a mutually reinforcing system that can exhibit multiple stable states.

Causal Loop Diagram (text representation):
Population (P) → (+) Resource Extraction → (-) Resource Base (R)
Resource Base → (+) Nutrition (N) → (+) Birth Rate → (+) Population
Resource Base → (-) Mortality → (+) Population
Cooperation (C) → (-) Wasteful Extraction → (+) Resource Base
Population Density → (∩) Cooperation (inverted-U: too low or too high reduces cooperation)
Resource Scarcity → (-) Cooperation (conflict increases)
Nutrition → (+) Cooperation (well-fed people cooperate more)
Cooperation → (+) Allomothering → (-) Infant Mortality → (+) Population

3. Full System of Differential Equations

3.1 Resource Base Dynamics

\[ \dot{R} = r_R R\left(1 - \frac{R}{K}\right) - \frac{\epsilon_0 P}{1 + \alpha C} \]

Where \(r_R\) is intrinsic regeneration rate, \(K\) is carrying capacity, \(\epsilon_0\) is baseline extraction per capita, and \(\alpha\) is cooperation efficiency.

3.2 Carrying Capacity Degradation

\[ \dot{K} = -\delta_K \cdot \max(0,\; E - r_R R) + \rho_K (K_{max} - K) \]

\(E = \epsilon_0 P/(1+\alpha C)\) is extraction rate. Carrying capacity degrades when extraction exceeds regeneration and recovers toward \(K_{max}\).

3.3 Population Dynamics (Age-Structured)

\[ \dot{P} = \beta(N,C) \cdot F - \mu(N,R) \cdot P - \sigma P \]

Where \(F \approx 0.4P\) is reproductive-age females (15-45 years). Birth rate \(\beta\) depends on inter-birth interval:

\[ IBI = IBI_{min} + (IBI_{max} - IBI_{min})(1-N)(1-C) \] \[ \beta = \frac{1}{IBI} \]

Mortality \(\mu\) follows a sigmoid threshold function:

\[ \mu(N,R) = \mu_0 + \frac{\mu_1}{1 + \exp(k_\mu(N - N^*))} \]

3.4 Nutritional Adequacy

\[ \dot{N} = \frac{1}{\tau_N}\left[\frac{\eta C R}{P \cdot \theta(F)} - N\right] \]

With demand weighting \(\theta(F) = \theta_0\left(1 + \psi \frac{\beta F}{P}\right)\) accounting for pregnancy/lactation costs.

3.5 Social Cooperation Capital

\[ \dot{C} = c_0 N P \max\left(0,\;1 - \frac{P}{P_{crit}}\right) - \delta_C C - \kappa C \max\left(0,\;1 - \frac{R}{R_{crit}}\right)^2 \]

Cooperation forms at intermediate population densities and collapses under resource scarcity.

3.6 Resource Quality Dynamics

\[ \dot{\eta} = -\delta_\eta (1-C)\frac{E}{R} + \rho_\eta(\eta_{max} - \eta) \]

Over-extraction with low cooperation depletes high-quality resources (fatty fish, large game).

4. Stochastic Processes

4.1 Monsoon Failure

Based on historical Indian Meteorological Department data (1870-2020):

\[ P(\text{failure}) = 0.15 \quad \text{per year} \]

When failure occurs, parameters are scaled for that year:

\[ r_R^{\text{eff}} = r_R \times 0.7,\quad K^{\text{eff}} = K \times 0.8,\quad \eta^{\text{eff}} = \eta \times 0.85 \]

4.2 Random Births

Poisson-distributed noise added to annual births:

\[ B_{\text{extra}} \sim \text{Poisson}(\lambda = \beta F \cdot 0.15 \cdot \xi) \]

Where \(\xi \sim \text{Uniform}(0.6, 1.4)\) represents annual fertility variation.

5. Age-Structured Population (Discrete)

The population is tracked in 5-year age bins (0-4 to 80+) using a Leslie matrix approach:

\[ \mathbf{a}_{t+1} = \mathbf{L}_t \cdot \mathbf{a}_t \]

Where \(\mathbf{L}_t\) incorporates age-specific mortality rates \(m_a\) and fertility rates \(f_a\). Mortality follows a typical pre-industrial profile:

7. Numerical Methods

The system is solved using a hybrid approach:

  • Continuous dynamics: 4th-order Runge-Kutta (RK45) with adaptive step size, integrated quarterly (dt = 0.25 years).
  • Discrete events: Monsoon failures and random births applied at year boundaries.
  • Age structure: Updated annually using Leslie matrix with age-specific mortality.
  • State clamping: Variables bounded to prevent numerical instability (\(R \ge 50\), \(N \in [0,1]\), etc.).

8. Expected System Behaviors

The model exhibits three qualitative attractors depending on parameter regimes:

  1. Stable subsistence equilibrium: Population stabilizes at 400-600 persons, cooperation moderate, resources sustainable.
  2. Cooperation-collapse / Malthusian trap: \(C \to 0\), \(N\) crashes, population overshoots then collapses.
  3. Ecological exhaustion: \(K\) degrades irreversibly, village becomes nonviable (\(P \to 0\)).
